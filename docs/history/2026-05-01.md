# 2026-05-01 — Erledigte Aufgaben

## Dashboard – „No links yet"-Bug

**Problem:** Das Dashboard zeigte „No links yet", obwohl Links gespeichert waren.  
**Ursache:** `getPublishStatistics()` nutzt einen Transient (`linkdigest_publish_stats`), der nach Änderungen nicht invalidiert wurde.  
**Fix:** `delete_transient('linkdigest_publish_stats')` in drei Mutationspfaden ergänzt:
- `src/php/traits/RestApi.php` → `restAddLink()` und `restDeleteLink()`
- `src/php/traits/Admin/AddLink.php` → `insertNewLink()`

---

## Dashboard – Recent Unpublished Links: Kompakteres Layout

- **Papierkorb-Icon verdoppelt** (16 px → 36 px) via `.lb-delete-btn .dashicons`
- **Layout-Sprung beim Löschen behoben:** Beim Klick auf den Papierkorb sprang der Inhalt nach oben, weil der Button im Flex-Header die Höhe bestimmte.  
  **Lösung:** HTML-Struktur umgebaut – Button als direktes `<li>`-Kind (eigene Grid-Spalte), nicht mehr im `.lb-link-item-header`.
- **Whitespace reduziert:** `<li class="lb-link-item">` auf CSS-Grid umgestellt (`grid-template-columns: 1fr auto`, `grid-template-rows: auto auto auto`), Padding von 8/12 auf 5/12 px reduziert.

**Geänderte Dateien:** `src/php/traits/Admin/Dashboard.php`, `dashboard.css`

## Dashboard – Untertitel im Boxheading „Recent Unpublished Links"

Hinter dem Box-Titel wird nun der nächste Veröffentlichungszeitpunkt bzw. der Fortschritt bis zur Veröffentlichung angezeigt.

- **Zeitbasierte Modi** (daily, weekly, monthly, age): Kalender-Icon + „next: 1. Jun. 2026, 09:00 Uhr"
- **Count-Modus**: „2 out of 10 left until publish" (ohne Icon)
- **Manual-Modus**: kein Untertitel

Neue Methode `unpublishedLinksSubtitle(): array` in `src/php/traits/Admin/Dashboard.php` gibt `['text' => string, 'icon' => string]` zurück.

`.lb-box-subtitle` und `.lb-box-subtitle .dashicons` in `dashboard.css` ergänzt.
